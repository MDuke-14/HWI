<product_requirements>
The application is a comprehensive employee time-management and reporting system for HWI Unipessoal.

**Core Functionality:**
*   Employee clock-in/out with location tracking.
*   PWA support for mobile access.
*   Vacation and absence requests with admin approval and email notifications.

**Reporting System:**
*   A Relatórios section provides detailed time reports.
*   Admins can view reports for any user.
*   Reports can be generated for standard monthly periods or custom date ranges.
*   Reports must correctly calculate Trabalho Suplementar (Overtime) and Trabalho Suplementar Dias de Descanso (Special Day Hours).
*   Days off, approved vacations, and absences (unjustified non-working days) must be clearly marked in reports and corresponding PDFs.
*   Subsidy day counts (e.g., meal allowance) are tracked, but monetary values have been removed. For special days, a meal allowance is only granted if the user works 5+ hours.
*   PDFs of reports must be downloadable, reflecting the selected user and date range.

**OTs - Ordens de Trabalho (formerly Technical Reports):**
*   A separate system for creating and managing work orders, accessible via a floating action button.
*   Includes full CRUD functionality for clients and OTs.
*   Permissions are tiered: all authenticated users can create and edit OTs and their details (technicians), but only admins can delete clients or OTs.

**Admin-Specific Features:**
*   A real-time status dashboard to view all employees' current activity (clocked-in, on break, vacation, etc.) for the day.
*   Ability to manually edit any user's time entries and set a day's status to FALTA (Absence), FÉRIAS (Vacation), or FOLGA (Day Off).
</product_requirements>
