**Key Technical Concepts**
-   **Background Scheduled Jobs (APScheduler):** Using a library to run tasks (like checking for missed clock-ins) at specific, recurring times.
-   **Asynchronous Emailing (aiosmtplib):** Sending emails from the backend without blocking the main application thread.
-   **Token-Based URL Authorization:** Generating and managing secure, single-use tokens for actions initiated from external systems (like email).
-   **Environment Configuration:** Separating backend and frontend configuration and ensuring the backend has access to necessary frontend URLs for generating correct links.
-   **Complex Form Logic:** Managing state for dynamic forms with calculated fields (e.g., Kms calculation, total minutes from HH:MM).
