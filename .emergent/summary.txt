<product_requirements>
The application is a comprehensive employee time-management and Work Order (OT - Ordem de Trabalho) system.

**Core Functionality:**
*   Employee clock-in/out.
*   Admin dashboard for real-time monitoring and custom reporting.
*   Admins can download monthly PDF reports for any user directly from the dashboard.

**OTs - Ordens de Trabalho System:**
This is a full-fledged work order management module with a complex set of interconnected features.
*   **Entities:** Manages Clients, OTs, Equipment, and Interventions.
*   **Equipment DB:** An equipment database is maintained per client. Equipment is automatically created or updated when a new OT is made. OTs can be created by selecting from a client's existing equipment.
*   **Interventions:** OTs can have multiple dated interventions, each with a motivo (reason) and relatório (report), allowing for a detailed history of a single work order.
*   **Technicians:** Technicians (system users) can be added to OTs. The date of assignment is recorded. The work-hour code (e.g., day, night, weekend) is automatically suggested based on the date but can be manually overridden via a popup.
*   **Status Management:** OTs have a multi-stage status (Orçamento, Em Execução, Concluído, Facturado). Status changes are made via a popup, with the Facturado status being restricted to admins.
*   **Search:** A dedicated tab allows for filtering all OTs by their current status.
</product_requirements>
